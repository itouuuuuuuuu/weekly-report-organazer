# weekly-report-organazer
ただのキッチンタイマーです。<br>
発表者表示機能あります。

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project for local
$ npm run generate

# generate static project for production
$ npm run generate:production
```
